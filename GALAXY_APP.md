# 갤럭시 앱 설치

우리 가족 앱은 Galaxy에서 Samsung Internet 또는 Chrome을 통해 설치할 수 있는 PWA입니다. 별도 APK 없이 앱 화면에 설치되며, 설치 후에는 주소창 없는 독립 화면으로 실행됩니다.

## 앱에서 설치

1. 배포된 우리 가족 사이트를 Galaxy에서 엽니다.
2. 로그인 후 **설정** 탭으로 이동합니다.
3. **갤럭시 앱 설치** 카드에서 **앱 설치**를 누릅니다.
4. 설치 확인 창에서 승인합니다.

설치 버튼이 바로 표시되지 않으면 아래 브라우저 메뉴를 사용합니다.

- **Samsung Internet:** 상단 설치 아이콘 또는 메뉴의 **앱 설치**
- **Chrome:** 메뉴(⋮)의 **홈 화면에 추가** 또는 **앱 설치**

## 설치 후 동작

- 앱 화면에서 `우리 가족` 아이콘으로 실행
- 브라우저 주소창 없는 standalone 화면
- 한 번 열어 본 앱 화면과 정적 리소스의 오프라인 재실행
- 기존 일정 브리핑 및 알림 기능 유지
- 새 배포본 확인 시 서비스 워커 캐시 자동 교체

## 확인 항목

1. 앱 아이콘으로 실행했을 때 주소창이 보이지 않는지 확인합니다.
2. 설정의 **갤럭시 앱 설치** 카드가 `설치 완료` 상태인지 확인합니다.
3. 온라인에서 한 번 실행한 뒤 비행기 모드에서 다시 열어 기본 화면이 표시되는지 확인합니다.
4. 설정에서 일정 브리핑 테스트 알림을 실행해 푸시가 유지되는지 확인합니다.
