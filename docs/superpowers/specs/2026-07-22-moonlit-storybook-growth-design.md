# Family 별빛 동화책 성장 화면 설계

## 결정

성장 화면의 2차 시각 방향은 사용자가 선택한 `C. 별빛 동화책`으로 한다. 폰트는 제목과 본문을 나누지 않고 앱 전체에서 `SUIT Variable` 하나만 사용한다. 동화책 감성은 폰트 혼용이 아니라 색감, 카드 표면, 선과 여백으로 만든다. 이 문서의 단일 폰트 결정은 1차 타이포그래피 설계의 MaruBuri·SUIT 혼용 결정을 대체한다.

## 목표

- 모든 화면의 제목, 본문, 버튼, 입력값, 날짜와 숫자가 한 폰트로 일관되게 보인다.
- 별빛 밤의 성장 화면이 지금보다 확실히 감성적인 동화책처럼 보인다.
- 육아 기록의 정보 밀도와 빠른 조작성을 유지한다.

## 단일 폰트 시스템

- 앱의 유일한 글꼴 패밀리는 저장소에 자체 호스팅한 `SUIT Variable`이다.
- 제목은 SUIT 700, 섹션 제목은 650~700, 본문은 400, 버튼과 강조는 700을 사용한다.
- MaruBuri `@font-face`, display 전용 변수, 런타임 적용 규칙, WOFF2 자산과 별도 라이선스 파일을 제거한다.
- 이모지는 시스템 컬러 이모지 폰트를 사용하되 텍스트 글꼴 체계에는 포함하지 않는다.

## 별빛 동화책 시각 언어

### 배경

- 별빛 밤의 `#growthView`에 청록빛 수채화 번짐과 매우 옅은 별빛 점을 CSS 배경으로 추가한다.
- 배경은 텍스트 뒤에서 움직이지 않으며 애니메이션을 사용하지 않는다.
- 장식은 콘텐츠 대비를 해치지 않도록 낮은 불투명도로 제한한다.

### 카드

- 아기 프로필, 기록하기, 돌봄 패턴, AI 육아 도우미 카드를 반투명한 밤색 양피지 표면처럼 만든다.
- 금빛과 청록빛을 섞은 얇은 테두리, 내부 하이라이트, 부드러운 그림자를 사용한다.
- 카드마다 모서리 비율을 조금 다르게 해 손으로 자른 종이처럼 보이게 하되 레이아웃 경계는 유지한다.

### 버튼과 구분선

- 수유와 기저귀 빠른 기록 버튼은 서로 다른 은은한 빛의 스티커 카드로 구분한다.
- 활성 탭과 주요 행동은 기존 의미 색을 유지하며 장식 때문에 상태 구분이 흐려지지 않게 한다.
- 주요 섹션 사이에는 pseudo-element로 만든 얇은 금빛 별자리형 구분선을 사용한다.
- 버튼 문구와 접근 가능한 이름은 바꾸지 않는다.

## 테마 범위

- 강한 별빛 표현은 `html[data-family-theme="night"] #growthView`에만 적용한다.
- 포근한 숲 등 밝은 테마에는 단일 SUIT 폰트만 공통 적용하고 기존 카드 색과 대비를 유지한다.
- 일정, 동화, 요청, 설정 화면은 폰트 통일 외에 이번 단계에서 재디자인하지 않는다.

## 접근성과 동작 경계

- 입력 폼 구조, 데이터, AI 답변, Supabase, 인증, 기록 버튼 동작을 변경하지 않는다.
- 의료 안전 문구, 날짜, 수유량, 차트 축에는 장식 이모지를 추가하지 않는다.
- 기존 focus-visible, 터치 크기, 텍스트 대비와 화면 확대 정책을 유지한다.
- `prefers-reduced-motion`에 의존할 새 애니메이션은 만들지 않는다.

## 구현 위치

- `typography-system.css`: SUIT 단일 패밀리와 굵기 역할
- `night-theme-polish.css`: 별빛 밤 성장 화면의 배경, 카드, 버튼과 구분선
- `config.js`: 변경된 CSS 모듈 버전 갱신
- `test/`: 단일 폰트와 스타일 범위에 대한 회귀 계약

새 화면별 CSS 파일이나 JavaScript 동작은 추가하지 않는다.

## 검증

- 계약 테스트에서 MaruBuri 런타임 선언과 자산이 제거되고 SUIT만 남았는지 확인한다.
- `npm test`, `npm run check`, `git diff --check`를 통과한다.
- 로그인된 브라우저에서 별빛 밤과 대표 밝은 테마를 각각 확인한다.
- 390×844, 430×932, 768×1024, 1440×900에서 가로 넘침, 카드 잘림, 버튼 겹침을 확인한다.
- 제목, 본문, 버튼의 computed `font-family`가 모두 `Family UI`인지 확인한다.

## 범위 밖

- 고정 성장 기록 버튼 위치 변경
- AI 전략 본문 글자 크기 개선
- 일정·동화·요청·설정 카드의 전면 재디자인
- 새 이미지 생성 또는 외부 이미지 다운로드
- Supabase 배포와 GitHub Pages 배포

## 성공 기준

- 사용자가 화면 어디에서도 서로 다른 한글 글꼴이 섞였다고 느끼지 않는다.
- 별빛 밤 성장 화면은 기존보다 명확하게 동화책처럼 보이지만 기록과 입력은 빠르게 읽힌다.
- 다른 테마와 기존 기능에 시각적·동작적 회귀가 없다.
